# Benchmarks

This directory contains contest benchmarks. Each benchmark folder has the following folders:

* **EDA_files**: This folder has all the provided EDA files of respective design.
   * `<design_name>.v`: This is the legally placed netlist generated using ASAP7 library.
   * `<design_name>.def`: This is the given def for the design.
   * `<design_name>.sdc`: This is the given SDC constraint file for the design.
   * `node.csv, nets.csv, metrics.csv`: These are the files containing the node, net and design-level metrics for the given benchmarks. node.csv & nets.csv are generated using `write_node_and_net_files` proc from evaluation/validity_check/OpenROAD_utils.tcl. metrics.csv is generated using parse_log.py script in evaluation folder. These files are provided to check the validity of the output db files generated by the contestants' solution and to compute the score for the solution comparing with the baseline metrics.
