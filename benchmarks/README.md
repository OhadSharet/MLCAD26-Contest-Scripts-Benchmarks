# Benchmarks

This directory contains contest benchmarks. Each benchmark folder has the following folders:

* **EDA_files**: This folder has all the provided EDA files of respective design.
   * `<design_name>.v`: This is the legally placed netlist generated using ASAP7 library.
   * `<design_name>.def`: This is the given def for the design.
   * `<design_name>.sdc`: This is the given SDC constraint file for the design.
   * `node.csv`:  Contains name of node, type of node, and (x,y) coordinates of that each node. 
   * `nets.csv`: Shows Net information for the benchmark.
   * `metrics.csv`: Contains information through which baseline scores are calculated. 
   
   `node.csv`, `nets.csv` and `metrics.csv` are provided to check validit of output files generated by contestants' solution and to compute score for baseline solution. 
---

## Directory Structure

```
benchmarks/
└── <design_name>/
   ├── <design_name>.v
   ├── <design_name>.def
   ├── <design_name>.sdc
   ├── metrics.csv
   ├── node.csv
   └── nets.csv
```
